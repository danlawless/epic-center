# Program Overview

> *Structured journeys for deep transformation*

---

## Philosophy

Our programs are not packages - they are carefully designed containers for transformation. Each duration serves a specific purpose, and the right choice depends on the guest's needs, history, and goals.

**Core Truth**: Transformation cannot be rushed. We offer different lengths not for convenience, but because different depths of work require different amounts of time.

## Choosing a Program

### Assessment Criteria

**Duration Recommendations Based On:**

| Factor | Shorter (7-14) | Longer (21-28) |
|--------|----------------|----------------|
| First experience | Consider 7-14 | Only if ready |
| Trauma history | Simple/processed | Complex/unprocessed |
| Life demands | Cannot take time | Can take time |
| Previous medicine work | Experienced | Limited |
| Support system | Strong | Building |
| Goals | Specific | Comprehensive |
| Budget | Limited | Available |
| Energy/health | Lower | Sufficient |

### Consultation Process

1. **Initial Application** - Basic information
2. **Screening Call** - Understand needs and history
3. **Medical Review** - Ensure safety
4. **Recommendation** - Suggest appropriate program
5. **Guest Decision** - Final choice with guidance
